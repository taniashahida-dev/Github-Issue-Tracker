1️⃣ What is the difference between var, let, and const?
=> Var : The old way to declare variables. It is function-scoped and can be redeclared.
   Let :  The modern way to declare variables. It is block-scoped  and allows the value to be changed later.
   Const :  Used for fixed values. It is block-scoped and cannot be changed or redeclared once set.

2️⃣ What is the spread operator (...)?
=> The spread operator uses three dots  to unpack or spread elements from an array or object into a new one. It is commonly used to copy arrays or combine multiple lists into one.

3️⃣ What is the difference between map(), filter(), and forEach()?
=> Map() : Runs a function on every item in an array and returns a new array with the transformed values.
   Filter() : Checks every item against a condition and returns a new array containing only the items that pass.
   forEach() : Loops through every item to perform an action (like printing) but does not return a new array.

4️⃣ What is an arrow function?
=> An arrow function is a shorter syntax for writing functions in JavaScript. It removes the need for the function keyword and provides a cleaner way to write code, especially for simple, one-line tasks.

5️⃣ What are template literals?
=> Template literals allow you to create strings using backticks (`) instead of quotes. By using this we can insert variables directly into the text using the ${variable} syntax.
